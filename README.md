# 🌌 Portafolio Personal Inmersivo - Alejandro Herrera

Este es mi portafolio web personal, diseñado para mostrar mis proyectos de ingeniería en un entorno visual dinámico e interactivo. El proyecto utiliza una estética de "espacio profundo" combinada con técnicas modernas de diseño web.

## 🚀 Características Principales

* **Diseño Responsivo:** Implementado con **CSS Flexbox** para asegurar que los bloques de proyectos se adapten a diferentes tamaños de pantalla.
* **Estética Glassmorphism:** Los bloques de contenido utilizan transparencias y desenfoques (`backdrop-filter: blur`) para resaltar sobre el fondo sin perder legibilidad.
* **Fondo Dinámico:** Integración de un video de alta definición como fondo fijo para una experiencia de usuario inmersiva.
* **Sección de Proyectos:** Organización modular de mis desarrollos en **Java** y **C++**.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos personalizados, animaciones y diseño de rejilla flexible.
* **Media Queries:** Para la adaptabilidad en dispositivos móviles.

## 📂 Estructura del Proyecto

* `index.html`: Estructura principal de la página.
* `MipaginaStyle.css`: Hoja de estilos con efectos de transparencia y flexbox.
* `videoFondo.mp4`: Recurso multimedia para el ambiente espacial.

## 🖥️ Proyectos Destacados

1.  **Sistema de Gestión para Clínica Veterinaria (Java):** Aplicación orientada a objetos para la gestión de pacientes y consultas.
2.  **Sistema de Registro de Productos (C++):** Manejo de estructuras y archivos binarios para inventarios.
3.  **Agenda de Contactos (C++):** Sistema de persistencia de datos y manipulación de archivos de texto.
